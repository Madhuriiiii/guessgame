# 🎯 Guess The Number Game

A simple JavaScript-based number guessing game.

## 📌 About The Project

This is a beginner-friendly game built using JavaScript.  
The user enters a maximum number, and the program generates a random number between 1 and that maximum.

The player keeps guessing until:
- They guess the correct number 🎉
- Or type "quit" to exit the game

## 🚀 Features

- Random number generation
- Input validation
- Infinite loop until correct guess
- Quit option
- Console-based interaction

## 🛠️ Built With

- HTML
- JavaScript

## ▶️ How To Run

1. Open the project in a browser
2. Open Developer Tools → Console
3. Follow the prompts
